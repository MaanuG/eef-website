# Empowering Education Foundation Website

The Empowering Education Foundation (EEF) is a student-led nonprofit organization dedicated to bridging educational gaps through accessible tutoring, cultural exchange, and leadership development.

This repository contains the official source code for the EEF website, built using HTML, CSS, and JavaScript. The site showcases our mission, initiatives, curriculum, team, gallery, volunteer opportunities, and contact resources through a modern, interactive user experience.

## Features

- ☀️ Custom animated hero section
- 🌍 Interactive global initiatives map
- 🎓 Team and leadership profiles
- 📚 Curriculum and educational programs
- 🖼️ Interactive photo gallery
- ❤️ Causes and impact sections
- 📈 Animated statistics counters
- 🙋 Volunteer application form
- ✉️ Contact form integration
- 📱 Fully responsive design
- ✨ Scroll-triggered animations and transitions

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Apps Script (Google Forms/Sheets integration)
- Font Awesome
- Google Fonts (Inter)

## Organization

Empowering Education Foundation is a nonprofit organization founded by students from Troy High School in Fullerton, California, working to connect volunteers with students around the world through education and cultural exchange.
